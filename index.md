# Privacy Policy

## Overview
This application integrates with Atlassian products using OAuth 2.0 (3LO).

## Data Access
The app accesses only the data explicitly authorized by the user.

## Data Usage
Data is used solely to provide application functionality.

## Data Storage
The app does not permanently store personal data unless required.

## Data Sharing
No data is shared with third parties.

## User Control
Users can revoke access at any time from their Atlassian account.
